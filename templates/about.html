ℹ️ About This Project

📋 Project Overview

This is a Text Emotion Classification system that uses Deep Learning to detect emotions from text input. The model is trained on a dataset of text samples labeled with different emotions.

🤖 Model Architecture

  • Embedding Layer: Converts words to dense vectors of size 128
  • Flatten Layer: Flattens the embedding output
  • Dense Layer: 128 units with ReLU activation
  • Dropout Layer: 50% dropout for regularization
  • Output Layer: Softmax activation for multi-class classification

💡 How It Works

  1. Enter your text in the input field
  2. The text is tokenized and converted to sequences
  3. The model processes the text through neural network layers
  4. The model outputs probability scores for each emotion
  5. The emotion with the highest probability is displayed along with confidence scores
