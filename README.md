# 🎭 Text Emotion Classification

A deep learning-based web application that detects emotions from text input. The model is trained on a dataset of text samples and can classify emotions such as joy, sadness, anger, fear, love, and surprise with high accuracy.

## ✨ Features

- **Real-time Emotion Detection**: Instantly analyze text and detect emotions
- **Interactive Web Interface**: Clean, modern UI with Bootstrap 5
- **Confidence Scores**: View probability scores for all emotions
- **Visual Feedback**: Progress bars showing confidence levels for each emotion
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **REST API**: Programmatic access to emotion prediction
- **Model Information**: Detailed about page with model architecture and performance metrics

## 🛠️ Technologies Used

### Backend
- **Python 3.8+**: Core programming language
- **Flask 2.3.2**: Web framework for serving the application
- **TensorFlow 2.13.0**: Deep learning framework for model training and inference
- **Keras**: High-level neural networks API
- **Scikit-learn**: For label encoding and data preprocessing

### Frontend
- **HTML5/CSS3**: Structure and styling
- **Bootstrap 5**: Responsive UI components
- **JavaScript**: Client-side interactivity and API calls

### Data Processing
- **Pandas**: Data manipulation and analysis
- **NumPy**: Numerical computing
- **Pickle**: Model serialization
